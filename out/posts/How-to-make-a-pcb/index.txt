f:T27f8,<h2>Getting Started with PCB Design and Image Editing</h2>
<p>To begin, you&#39;ll need a basic understanding of electronics and some image editing skills. Here&#39;s what you&#39;ll need:</p>
<h3>Downloads</h3>
<ol>
<li><strong><a href="https://www.kicad.org/">KiCad</a></strong>: Used for designing the PCB.</li>
<li><strong><a href="https://inkscape.org/">Inkscape</a></strong>: Used for processing the PCB design to make it compatible with CNC software.</li>
</ol>
<p>Additionally, you can use <strong>Photoshop</strong> for advanced editing, but I’ll be using <strong><a href="https://www.photopea.com/">Photopea</a></strong> a free, browser-based Photoshop alternative.</p>
<hr>
<h3>Drawing the Schematic</h3>
<p>Our schematic is simple, so we&#39;ll sketch it by hand first:</p>
<p><strong>Schematic Diagram:</strong></p>
<p><img src="/images/Pasted%20image%2020241226024552.png" alt=""></p>
<p><strong>Pin Details:</strong></p>
<ul>
<li><strong>M+, M-</strong>: Motor pins.</li>
<li><strong>5V</strong>: Positive 5 volts.</li>
<li><strong>GND</strong>: Ground connection.</li>
<li><strong>PWM</strong>: Signal from the Arduino.</li>
</ul>
<p><strong>Circuit Functionality:</strong></p>
<ul>
<li>This circuit controls motor speed using a PWM signal.</li>
<li>The <strong>transistor</strong> switches the motor&#39;s power source.</li>
<li>The <strong>diode</strong> protects the circuit from back EMF (electromotive force).</li>
<li>The <strong>resistor</strong> ensures that the base of the transistor receives the correct voltage to operate within its safe limits, avoiding damage to the transistor.</li>
<li>The <strong>NPN transistor</strong> allows the Arduino to provide a positive signal directly.</li>
</ul>
<hr>
<h3>Basics of KiCad</h3>
<p>After installing KiCad, open the software. You should see the following interface:</p>
<p><img src="/images/Pasted%20image%2020241226025807.png" alt=""></p>
<h4>Creating a New Project in KiCad</h4>
<ol>
<li><p>Navigate to <strong>File &gt; New Project</strong> or click the <strong>New Project</strong> icon in the toolbar.</p>
<p> <img src="/images/Pasted%20image%2020241226025931.png" alt=""></p>
</li>
<li><p>Select the location where you want to save your project and provide a name for it. Once you save, the following screen will appear:</p>
<p> <img src="/images/Pasted%20image%2020241226081252.png" alt=""></p>
</li>
</ol>
<h5>Understanding the New Files</h5>
<p>After creating the project, two important files will be generated:</p>
<ul>
<li><p><strong>.kicad_sch</strong>: This file contains the schematic diagram of the project.</p>
</li>
<li><p><strong>.kicad_pcb</strong>: This file contains the PCB layout of the project.</p>
<p>  <img src="/images/Pasted%20image%2020241226081419.png" alt=""></p>
</li>
</ul>
<p>We’ll begin by working on the schematic. Once it is completed, we’ll move on to the PCB layout.</p>
<hr>
<h4>Starting with the Schematic</h4>
<ol>
<li><p>Open the <strong>.kicad_sch</strong> file by double-clicking it. This will open the schematic editor, and the following screen will appear:</p>
<p> <img src="/images/Pasted%20image%2020241226083410.png" alt=""></p>
</li>
</ol>
<h5>Overview of the Schematic Editor Interface</h5>
<ol>
<li><p><strong>Toolbar (Box 1)</strong></p>
<ul>
<li>Provides shortcuts for common actions such as creating, saving, undoing/redoing, zooming, and running the electrical rules check (ERC).</li>
<li>Also allows you to switch between different editors (e.g., schematic, PCB, symbol, and footprint editors).</li>
</ul>
</li>
<li><p><strong>Tools Panel (Box 2)</strong></p>
<ul>
<li>Includes tools for placing components, wires, labels, power symbols, and other elements needed to create your schematic.</li>
<li>Contains additional features for managing schematic design, such as creating connections, adding text annotations, or drawing shapes.</li>
</ul>
</li>
</ol>
<p>Now, you’re ready to start building your schematic! Let’s dive in.</p>
<p>we need to know how the controls work </p>
<h4>Mouse Controls</h4>
<ul>
<li><p><strong>Left Click</strong>:<br>  Selects components, wires, or other elements.<br>  If used on an empty space, it deselects the current selection.</p>
</li>
<li><p><strong>Right Click</strong>:<br>  Opens a context menu with options relevant to the selected item or area.</p>
</li>
<li><p><strong>Scroll Wheel</strong>:<br>  Zooms in and out of the canvas.<br>  Hold the scroll wheel and move the mouse to pan around the canvas.</p>
</li>
</ul>
<hr>
<h3>Adding Components to the Schematic</h3>
<p>To add components, click on the following icon:<br><img src="/images/Pasted%20image%2020241226084528.png" alt=""></p>
<p>After a brief loading screen, this window will appear:<br><img src="/images/Pasted%20image%2020241226084706.png" alt=""></p>
<p>In the search field, type the name of the component you&#39;d like to add. For example, to add a resistor:</p>
<ul>
<li>Type <strong>R</strong> in the search field, and the resistor symbol will appear:<br>  <img src="/images/Pasted%20image%2020241226084855.png" alt=""></li>
<li>Select the resistor from the results, click <strong>OK</strong>, and place it on the canvas.</li>
</ul>
<p>Next, follow the same steps to add a diode.</p>
<hr>
<h3>Adding a Component Not Found in KiCad&#39;s Library</h3>
<p>For certain components, like the 2N2222 transistor, you may not find them in KiCad’s default library. Let’s go through the steps to add a new component to your library.</p>
<h4>Using SnapEDA to Download Components</h4>
<ol>
<li>Open <a href="https://www.snapeda.com/">SnapEDA</a> and search for the component by name:<br> <img src="/images/Pasted%20image%2020241226092718.png" alt=""></li>
<li>Select the result that matches the look of your desired component.</li>
</ol>
<p>Here’s what the component’s page looks like:<br><img src="/images/Pasted%20image%2020241226092920.png" alt=""></p>
<ul>
<li><strong>Symbol</strong>: Used in the schematic editor.</li>
<li><strong>Footprint</strong>: Used in the PCB layout.</li>
</ul>
<h4>Downloading the Symbol and Footprint</h4>
<ol>
<li>Click the <strong>Download Symbol and Footprint</strong> button. A window will pop up:<br> <img src="/images/Pasted%20image%2020241226101900.png" alt=""></li>
<li>Select <strong>KiCad</strong> as the format.</li>
</ol>
<blockquote>
<p><strong>Note</strong>: You will need to log in or create an account to continue.</p>
</blockquote>
<ol start="3">
<li><p>Choose the appropriate version. For example:<br> <img src="/images/Pasted%20image%2020241226103113.png" alt=""><br> If you&#39;re using KiCad version 6 or later, select <strong>V6 &amp; Later</strong>.</p>
</li>
<li><p>After downloading the <code>.zip</code> file, create a folder in your documents directory named <code>KiCad-lib</code>. This folder will store all your KiCad symbols and footprints.</p>
</li>
<li><p>Extract the downloaded file into the <code>KiCad-lib</code> folder.</p>
</li>
</ol>
<blockquote>
<p><strong>Why this step?</strong><br>Saving the files in a dedicated library folder ensures they remain accessible, even if the project folder is moved. Without this, KiCad may lose track of the symbols and footprints.</p>
</blockquote>
<hr>
<h3>Importing the Symbol into KiCad</h3>
<ol>
<li><p>Open KiCad and go to <strong>Preferences &gt; Manage Symbol Libraries</strong>:<br> <img src="/images/Pasted%20image%2020241229073019.png" alt=""></p>
</li>
<li><p>Click on <strong>Add existing library to table</strong>:<br> <img src="/images/Pasted%20image%2020241229073157.png" alt=""></p>
</li>
<li><p>Navigate to the folder where you extracted the file (<code>KiCad-lib</code>) and select the symbol file:<br> <img src="/images/Pasted%20image%2020241229073249.png" alt=""></p>
</li>
<li><p>The library will appear at the end of the table. Click <strong>OK</strong> to save the changes.</p>
</li>
</ol>
<hr>
<h3>Adding the Imported Component to the Schematic</h3>
<p>Now, when you add a symbol and search for <code>2N2222</code>, it will appear in the list:<br><img src="/images/Pasted%20image%2020241229073505.png" alt=""></p>
<p>Select it, and you’re ready to place the component on your schematic!</p>
